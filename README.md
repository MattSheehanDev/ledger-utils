# Prereqs

* **Visual Studio Code 1.6.1**
* **Nodejs v6.2.0**
* **NPM (installed with Nodejs) v3.8.9**


# Building ledger-utils
1. Make sure you are in the project directory
    ```
    > cd ~/sec-xbrl
    ```
2. Install the dependencies
    ```
    > npm install
    ```
3. Run the script
    ```
    > ./prices.js
    ```
